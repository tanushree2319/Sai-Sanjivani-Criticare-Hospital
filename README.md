# Sai Sanjivani Criticare Hospital | Bootstrap 5 + SASS Refactor

## 🎯 Project Overview

A complete architectural refactoring of the hospital landing page from single-file vanilla HTML/CSS to a modern, modular Bootstrap 5 + SASS architecture. This maintains the premium "Quiet Luxury" design identity while achieving:

- ✨ **Lightning-fast** performance (ultra-lightweight, minimal CSS payload)
- 🔧 **Highly modular** SASS structure for easy maintenance
- 📱 **Mobile-first** Bootstrap 5 grid system
- 🎨 **Perfect design preservation** of all custom transitions, glassmorphism, and editorial typography
- ♿ **Accessibility-first** HTML structure with semantic markup
- 🌐 **Enterprise-grade** build system with SASS compilation

## 🚀 Quick Start

### 1. **Installation**

```bash
npm install
```

This installs:
- **Bootstrap 5.3+** (lightweight grid essentials only)
- **SASS compiler** for production builds

### 2. **Development Mode** (Watch SASS changes)

```bash
npm run sass
```

This watches `scss/` directory and auto-compiles to `css/main.css`.

### 3. **Production Build** (Minified output)

```bash
npm run sass:build
```

Generates optimized, minified CSS ready for production.

### 4. **Local Server**

```bash
npm run serve
```

Starts a simple HTTP server on `http://localhost:8000` with live testing.

## 📊 SASS Architecture Breakdown

### **abstracts/_variables.scss**
Master theme configuration. Override Bootstrap defaults here:
- Color palette (primary, accent, emergency, text, backgrounds)
- Typography (font weights, sizes, line heights, letter spacing)
- Spacing scale ($spacing-xs through $spacing-xxl)
- Border radius, shadows, transitions
- Breakpoints for mobile-first approach

### **layout/_base.scss**
Foundation styles:
- HTML/body resets
- Container max-width (1500px) with responsive padding
- Heading defaults (H1-H6 sizing with clamp for fluidity)
- Paragraph and link styling
- Animation definitions
- Accessibility focus states

### **components/_header.scss**
Fixed header with premium morphing:
- Fixed positioning with z-index stacking
- Brand logo and text with hover color transitions
- Navigation centered layout with active states
- CTA buttons: Emergency (red), Call (outline), Book (white)
- Header:hover triggers complete color scheme shift

### **components/_buttons.scss**
All button variants:
- `.btn-main` - Primary action (context-aware: white on hero, blue on pages)
- `.btn-outline` - Secondary outline style
- Hover effects with lift transforms and subtle shadows
- Disabled states and loading animations

### **components/_badges.scss**
Floating elements:
- Hero badge (glassmorphism with blur)
- Verified/trust tags (green success style)
- Experience badge (floats on doctor image)
- Section subtitles and custom animations

### **sections/_hero.scss**
Hero section architecture:
- Full viewport height (100vh) with video background
- Dark gradient overlay (rgba layers for readability)
- Absolute centering with padding adjustments
- Hero content animation (fadeIn staggered)
- Subheading and Marathi box styling
- Action buttons group with responsive wrapping

### **sections/_trust-strip.scss**
Trust grid (4 items → 2 → 1):
- CSS Grid auto-fit layout
- Mobile-first: 1 column on mobile, 4 on desktop
- Hover lift effect on items
- Icon support (emoji or SVG)

### **sections/_about.scss**
2-column about section:
- CSS Grid (1fr 1fr) with responsive gap
- Image hover zoom effect
- Marathi text with colored left border
- Mobile reorders to image-top layout

### **sections/_stats.scss**
Counter metrics section:
- 3-column grid with gradient dividers
- Intersection observer triggers animation on scroll
- Smooth counter increment (2-second animation)
- Hover effects on stat items

### **sections/_doctor.scss**
Doctor profile section:
- 2-column flex/grid layout
- Image with floating "25+ Years" badge
- Verified medical tag styling
- Responsive stack on mobile
- Section subtitle and credentials

### **layout/_footer.scss**
Footer section:
- Dark primary background
- 3-column responsive grid
- Link hover underlines
- Newsletter form styling (optional)
- Social icons support
- Copyright section

## 🔧 Bootstrap 5 Integration (Lightweight)

Only essential Bootstrap modules imported in `main.scss`:

```scss
@import "../../node_modules/bootstrap/scss/functions";
@import "../../node_modules/bootstrap/scss/variables";
@import "../../node_modules/bootstrap/scss/mixins";
@import "../../node_modules/bootstrap/scss/root";
@import "../../node_modules/bootstrap/scss/reboot";
@import "../../node_modules/bootstrap/scss/containers";
@import "../../node_modules/bootstrap/scss/grid";
@import "../../node_modules/bootstrap/scss/utilities/*";
```

**NOT imported** (bloat reduction):
- Components (buttons, navs, modals, etc.)
- Forms
- Pagination
- Spinners

This keeps the final CSS payload **significantly lighter** than full Bootstrap while maintaining the powerful grid system.

## 🎯 Mobile-First Approach

All layouts use **mobile-first breakpoints**:

```scss
// Mobile first (default)
.element { /* Styles apply to 320px+ */ }

// Tablet and up (768px+)
@media (min-width: 768px) { }

// Desktop (992px+)
@media (min-width: 992px) { }

// Large desktop (1200px+)
@media (min-width: 1200px) { }
```

Responsive utilities are built into each component:
- `clamp()` for fluid typography
- `auto-fit` Grid for responsive columns
- Flexbox gaps that adjust per breakpoint

## 🎨 Key Design Features Explained

### **Header Morphing (Premium UX)**
```scss
.header {
  background: linear-gradient(to bottom, rgba(0,0,0,0.7) 0%, rgba(0,0,0,0) 100%);
  transition: all 0.4s ease;
  
  &:hover {
    background: rgba(255, 255, 255, 0.98);
    box-shadow: 0 10px 30px rgba(0, 0, 0, 0.1);
  }
}
```
Entire header smoothly transitions on hover for a luxury scanning experience.

### **Glassmorphism Effects**
```scss
backdrop-filter: blur(4px); // Creates frosted glass effect
background: rgba(255, 255, 255, 0.1); // Semi-transparent overlay
border: 1px solid rgba(255, 255, 255, 0.2); // Subtle edge
```
Used on badges, buttons, and overlay elements for premium, modern feel.

### **Responsive Typography with clamp()**
```scss
h1 { font-size: clamp(2rem, 5vw, 3.5rem); }
// Scales smoothly from 2rem (mobile) → 3.5rem (desktop) based on viewport
```

### **Counter Animation (Intersection Observer)**
Counters only animate when scrolled into view:
```javascript
const observer = new IntersectionObserver((entries) => {
  entries.forEach((entry) => {
    if (entry.isIntersecting) {
      animateCounter(entry.target);
      observer.unobserve(entry.target); // Only runs once
    }
  });
});
```

## 📝 Customization Guide

### **Change Primary Color**
Edit `scss/abstracts/_variables.scss`:
```scss
$color-primary: #0f4c75;        // Change this
$color-primary-dark: #0a314d;   // And this
```

### **Adjust Spacing**
Modify spacing scale in `_variables.scss`:
```scss
$spacing-md: 20px;  // All margin/padding adjusts automatically
```

### **Typography Changes**
Update font families and sizes:
```scss
$font-family-base: "Inter", sans-serif;
$font-size-base: 1rem;
```

### **Add New Section**
1. Create `scss/sections/_newsection.scss`
2. Import in `main.scss`
3. Use semantic HTML and SASS nesting
